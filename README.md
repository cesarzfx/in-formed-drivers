# in-formed-drivers

The form allows users to build a profile of their **ideal car** for a fictitious company, **Car Finders Inc.** The project emphasizes proper form structure, accessibility, field validation, layout organization using fieldsets, and responsive design.

---

## Page Overview

The webpage contains:

- A centered header with logo and branding
- A two-column responsive form layout
- Grouped sections using `<fieldset>` and `<legend>`
- Clean and modern styling

---

## Form Features

### Mechanical Specs
- Manufacturer (select dropdown)
- Body Type (radio buttons)
- Drive Options (checkboxes)
- Price Range (min and max number inputs)
- Transmission (radio buttons)
- Fuel Type (select dropdown)
- Year Range (number inputs)
- Mileage Preference (select dropdown)

### Personal Style
- Colour (color picker)
- License (text input)
- Interior Material (select dropdown)
- Exterior Add-Ons (checkboxes)
- Safety Features (checkboxes)
- Driver Assistance Level (radio buttons)
- Comments (textarea)

Required fields are visually marked with a red asterisk (*).
